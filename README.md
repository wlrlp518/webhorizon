## 需要注意的点

说实话,没有完美的服务商。用WebHorizon要留心这几点:

**线路不针对中国大陆优化**
如果你主要面向国内用户,要做好心理准备。晚高峰可能会慢,建议先买月付测试。

**部分机房可能下架**
历史上他们关停过智利、香港、日本的NAT VPS节点。虽然会帮迁移,但频繁换机房确实挺折腾。

**低配NAT VPS限制多**
256MB内存只能跑Debian 11这种轻量系统,Debian 12可能启动不了。而且NAT端口转发有限制,不适合复杂应用。

**退款政策严格**
只有48小时退款窗口,而且用了超过50GB流量、IP被拉黑、通过转让获得的服务都不能退。购买前想清楚。
